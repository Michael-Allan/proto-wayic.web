proto-wayic.web - Web interfacing conventions

This project documents the general, cross-project conventions that inform Web development and Web interfacing within the present wayic. It includes the common terminology, forms and configuration of HTML markup, CSS styling and JavaScript programming.

The files for this project are released under an MIT licence.
